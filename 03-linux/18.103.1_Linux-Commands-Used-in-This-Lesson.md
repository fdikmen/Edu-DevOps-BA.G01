# Derste Geçen Linux Komutları ve Anlamları

| Komut / Söz Dizimi | Açıklama |
|---|---|
| `echo $değişken` | Belirtilen ortam değişkeninin değerini ekrana yazdırır (örn. `echo $name`, `echo $PATH`, `echo $HISTFILE`) |
| `değişken=değer` | Shell içinde yeni bir değişken tanımlar (örn. `name="firat"`) — eşittir işaretinin etrafında boşluk olmamalı |
| `unset değişken` | Belirtilen değişkeni tamamen kaldırır (örn. `unset PS1`) |
| `env` | O anda mevcut olan tüm ortam değişkenlerini listeler |
| `export değişken=değer` | Bir değişkeni ortam değişkenine dönüştürür ve alt süreçlere (child process) aktarılmasını sağlar |
| `bash` | İçinde bulunulan shell'den yeni bir alt bash oturumu (child process) başlatır |
| `echo $$` | Mevcut bash sürecinin PID'sini (process ID) gösterir |
| `set` | Ortam değişkenlerini, fonksiyonları ve bash yapılandırmasındaki tüm ayarları listeler |
| `PATH=$PATH:/dizin/yolu` | Mevcut PATH değerinin sonuna yeni bir dizin ekler |
| `which komut` | Belirtilen komutun hangi kopyasının çalıştırılacağını gösterir |
| `whereis komut` | Sistemde bulunan komutun tüm kopyalarının konumlarını listeler |
| `whatis komut` | Komutun ne işe yaradığına dair kısa bir açıklama verir |
| `./program` | Bulunulan dizindeki bir programı doğrudan (göreli yol ile) çalıştırır |
| `/tam/yol/program` | Bir programı mutlak (tam) yol belirterek çalıştırır |
| `cat dosya` | Bir dosyanın içeriğini ekrana yazdırır (örn. `cat ~/.bash_history`) |
| `history` | Mevcut oturumda bellekte tutulan komut geçmişini numaralarla listeler |
| `!numara` | History listesindeki belirli numaralı komutu tekrar çalıştırır (örn. `!152`) |
| `!!` | En son çalıştırılan komutu tekrar çalıştırır |
| `!kelime` | Belirtilen kelimeyi içeren en son komutu bulup çalıştırır (örn. `!temp`) |
| `Ctrl + R` | Komut geçmişinde reverse-i-search (geriye doğru arama) modunu başlatır |
| `Ctrl + O` | Reverse-i-search modunda bulunan komutu onaylayıp çalıştırır |
| `↑` (yukarı ok) | Önceki komutları sırayla geri getirir |
| `HISTSIZE=0` | O oturum için komut geçmişi tutmayı devre dışı bırakır |
| `exit` | Mevcut shell oturumundan çıkar |
| `Ctrl + D` | Shell oturumunu sonlandırmak için `exit` ile aynı işlevi görür |
| `cd /dizin` | Belirtilen dizine geçiş yapar (örn. `cd /tmp`) |
| `mkdir dizin` | Yeni bir dizin (klasör) oluşturur |
| `pwd` | Bulunulan dizinin tam yolunu gösterir |
| `ls` | Bulunulan dizindeki dosya ve klasörleri listeler |

## Öne Çıkan Ortam Değişkenleri

| Değişken | Anlamı |
|---|---|
| `SHELL` | Kullanılan shell türü (örn. bash) |
| `HOSTNAME` | Sistemin makine adı |
| `EDITOR` | Varsayılan metin editörü |
| `PWD` | O anda bulunulan dizin |
| `USER` / `LOGNAME` | Oturum açan kullanıcının adı |
| `PATH` | Komut aramasında taranan, `:` ile ayrılmış dizin listesi |
| `HISTFILE` | Komut geçmişinin kaydedildiği dosyanın yolu (genelde `~/.bash_history`) |
| `PS1` | Komut istemi (prompt) tanımı |
| `OLDPWD` | Bir önceki bulunulan dizin |
| `HISTSIZE` | Bellekte tutulacak komut geçmişi sayısı |
