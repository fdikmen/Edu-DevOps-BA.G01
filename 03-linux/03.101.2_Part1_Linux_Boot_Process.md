## BIOS ve UEFI Firmware Sistemleri
  - BIOS (Basic Input/Output System)
      - MBR (Master Boot Record) => 512 byte
      - 2TiB den büyük=> X
  - UEFI (Unified Extensible Firmware Interface)
      - Hardware detaylarını görüntüleyebilen grafik arayüzü sunar.
      - 2 TiB üzerindeki diskleri ve GPT partition table'ı destekler.
      - EFI System Partition (ESP) üzerinden doğrudan kernel yükleyebilir.
      - Secure Boot ile imzasız yazılımların yüklenmesini engeller.
      - Network boot (PXE) gibi gelişmiş özellikleri destekler.

### Kernel Boot Parametreleri
| Parametre       | Açıklama ve Kullanım Amacı |
|----------------|----------------------------|
| `single` veya `s` | Sistemi single-user modda başlatır. Root parola sıfırlama ve recovery işlemleri için kullanılır. |
| `quiet` | Boot sırasında verbose kernel mesajlarının görüntülenmesini engeller. |
| `ro` | Root filesystem'i başlangıçta read-only olarak mount eder. |
| `rw` | Root filesystem'i başlangıçtan itibaren read-write modda mount eder. |
| `vga=792` | Belirli bir ekran çözünürlüğü ve renk derinliği zorlar. Display sorunlarını gidermek için kullanılır. |
| `init=/bin/bash` | Normal init yerine doğrudan bash shell başlatır. Recovery senaryolarında kullanılır. |
| `nomodeset` | Kernel-level graphics mode'u devre dışı bırakır. Display driver sorunlarını gidermek için kullanılır. |

## Tam Boot Akışı

| Aşama | Gerçekleşen İşlem |
|------|-------------------|
| 1. Power-On | Kullanıcı güç düğmesine basar; anakart elektriklenir. |
| 2. Firmware Başlar | BIOS veya UEFI firmware devreye girer, POST'u başlatır. |
| 3. POST | RAM, CPU ve diskler test edilir; hardware bütünlüğü doğrulanır. |
| 4. Bootloader Yüklenir | BIOS → MBR'dan; UEFI → ESP'den GRUB yüklenir. |
| 5. GRUB Menüsü | Kullanıcı kernel seçimini yapar ya da default otomatik seçilir. |
| 6. Kernel Yüklenir | GRUB seçilen kernel'i ve initramfs'i belleğe yükler. |
| 7. initramfs Açılır | Kernel, geçici RAM filesystem'i oluşturur; temel driver'lar yüklenir. |
| 8. Root FS Mount Edilir | Gerçek root filesystem disk üzerinden mount edilir. |
| 9. init Başlar | Kernel, PID 1 olarak init (systemd veya SysV) process'ini başlatır. |
| 10. Servisler Başlar | init tüm sistem servislerini, network'ü ve user session'larını başlatır. |
